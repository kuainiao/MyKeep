###使用方法
- 数据库连接
```PHP
    $db_host="localhost:3306";//连接的服务器地址
    $db_user="keep";//连接数据库的用户名
    $db_psw="mykeep";//连接数据库的密码
    $db_name="keep";//连接的数据库名称
```
